# Crossword Constructor - web

## Installation

1. Fork https://github.com/crossword-constructor/xword-web
1. clone forked repo
1. `$ mv env.example .env`
1. `$ yarn`
1. `$ cd client and yarn`
1. Make sure you have a mongo instance running, cd back to the root directory and...
1. `$ yarn dev`

## Technologies used

- Node
- Apollo Server
- Apollo Client
- GraphQl
- MongoDB
- React
