    Crossword Constructor - web

    Installation

    1. Fork https://github.com/crossword-constructor/xword-web
    2. clone forked repo
    3. $ mv env.example .env
    4. $ yarn
    5. $ cd client and yarn
    6. Make sure you have a mongo instance running, cd back to the root directory and...
    7. $ yarn dev
